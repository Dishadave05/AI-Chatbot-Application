# Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Chatbot-Application.git
```

2. Navigate to the project folder

```bash
cd AI-Chatbot-Application
```

3. Install dependencies

```bash
npm install
```

4. Start the application

```bash
npm start
```
